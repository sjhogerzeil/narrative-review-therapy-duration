# Layer 3: Counterpoint — accelerated and brief models

The strongest claims that deep change can happen faster. Engaged honestly, not dismissed.

## Models to examine

- **AEDP** (Diana Fosha) — accelerated experiential dynamic psychotherapy
- **ISTDP** (Davanloo / Frederickson) — intensive short-term dynamic psychotherapy
- **EMDR** — claims beyond symptom reduction
- **Ketamine/MDMA-assisted therapy** — accelerated trauma processing
- **Intensive formats** — concentrated sessions, multi-day retreats

## Annotation questions for each model

1. What population are they actually studying?
2. What do they define as the outcome?
3. What is the evidence base?
4. What is the strongest version of their argument?
5. Under what conditions could this model be right — and what would that mean for the working thesis?
6. Where are the genuine limits of applicability for the population in this review?
7. What can the long-duration position learn from this model?

## Saturation note

Layer 3 now has 11 annotated sources covering 5 counterpoint models. The EMDR cluster (3 sources) provides the strongest counterpoint, with the de Jongh 2024 state-of-science review as the anchor. The MDMA/ketamine cluster (3 sources) provides provocative but methodologically weaker evidence. The intensive formats review provides strong evidence on the delivery question. AEDP-specific sources are still missing (the EDT meta includes AEDP but no standalone AEDP study has been annotated). Consider adding: AEDP-specific outcome study, EDT follow-up advantage mechanism study, and a strong critique of EMDR claims for CPTSD from the phase-based treatment camp (e.g., Cloitre response).
